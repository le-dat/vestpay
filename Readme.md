## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/)

### Installation

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/your-repo/vestpay.git
    cd vestpay
    ```

2.  **Setup Front-end**:

    ```bash
    cd front-end
    pnpm install
    pnpm dev
    ```

3.  **Setup Backend**:
    ```bash
    cd ../backend
    pnpm install
    pnpm run start:dev
    ```
